class 속성

tag 들을 그룹으로 묶어서 공통된 속성을 적용하는 방법이다

여거 tag에 같은 이름의 class를 지정하고

style에 속성을 설정하면 공통된 모양을 만들수 있다.

style sheet에 지정하는 방법

tag 이름

tag 이름에 속성을 설정하면 html 문서 내의 모든 tag에 같은 모양이 적용된다.

#id

tag에 id 속성을 지정하면 특정한 1개의 tag에만 모양을 지정 할 수 있다.

id 속성을 PK 적인 성질을 갖고 있어서 한 html 문서내에서는 중복하여 같은 이름을 지정하면 안된다.

.class

tag에 class 속성을 지정하면 여러개의 tag에 공통된 속성을 지정 할 수 있다.

종류가 다른 tag에 같은 이름의 class를 지정 할 경우 tag.class 형식으로 지정하면 특정 tag에 설정된 class 들만 모양을 지정할 수 있다.
